# Daniel Mabalot Portfolio

Public portfolio site for Daniel Mabalot, built as a static GitHub Pages user site.

Live URL: https://dnyeldev.github.io/

## Site Structure

- `index.html` - portfolio homepage
- `case-law-firm-content-os.html` - content operations case study
- `case-ai-lead-intake.html` - lead intake case study
- `case-ai-workflow-dashboard.html` - workflow dashboard case study
- `styles.css` - visual system and responsive layout
- `script.js` - navigation, reveal states, and interaction polish
- `robots.txt` - crawler access and sitemap reference
- `sitemap.xml` - public page index
